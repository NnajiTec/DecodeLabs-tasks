# Project 2: Exploratory Data Analysis (EDA)

## Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a cleaned sales dataset. The objective was to understand the dataset, identify patterns, summarize key statistics, detect outliers, and generate business insights to support decision-making.

## Objectives
- Explore the dataset structure
- Calculate descriptive statistics
- Identify trends and patterns
- Detect outliers
- Generate business insights
- Provide recommendations

## Dataset
- Total Records: 1,200
- Total Variables: 14
- Status: Cleaned and ready for analysis

## Key Findings
- Average order value is ₦1,053.97.
- Chair generated the highest revenue.
- June 2024 recorded the highest monthly revenue.
- Cancelled and returned orders accounted for a significant portion of all orders.
- Only the Total Price variable contained notable outliers.
