# DecodeLabs Data Analytics Internship

Welcome to my DecodeLabs Data Analytics Internship repository. This repository contains the projects I completed throughout the internship, demonstrating practical skills in data cleaning, exploratory data analysis (EDA), and SQL data analysis.

## About

During this internship, I worked with real-world datasets to develop essential data analytics skills. Each project builds on the previous one, following the typical data analysis workflow—from preparing raw data to extracting meaningful business insights.
