# SQL Data Analysis – Project 3

## Objective
Use SQL queries to extract business insights from a sales dataset.

## Skills Demonstrated
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()

## Database
SalesInsightsDB

## Table
dbo.Orders

## Key Analyses
- Displayed and filtered sales records
- Sorted sales by value
- Counted orders and products
- Calculated total revenue and average order value
- Analyzed revenue by product
- Analyzed orders by payment method
- Analyzed order status distribution

## Tools Used
- SQL Server Management Studio (SSMS)
- SQL Server
